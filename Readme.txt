		Velkommen til CodersTrust databasesystem

Før man kan køre SQL-Scripts - skal følgende programmer downloades:
- MySQL Community Server
- MySQL Workbench 

Følgende Quereis skal køres i denne rækkefølgen 1 til 3:
1) DDL-Queries (Databasen)
2) DDL-Queries (Values)
3) DML-Queries 

Tilføjelser: I vedhæftende fil (DDL Queries(SQL)), fremgår det i tabellen "Users",
at kolonnen "Reg_date" indeholder datatypen VARCHAR fremfor DATE. 
VARCHAR er benyttet, idet at datatypen DATE skabte problemer ved implementeringen af CSV filer.

		Welcome to CodersTrust databasesystem

Before launching the SQL script make sure you are using:
- MySQL Community Server
- MySQL Workbench

Make sure to follow these instructions in order 1 to 3:
1) Run DDL-Queries (Database)
2) Run DDL-Queries (Values)
3) Run DML-Queries

Additions: In the file (DDL-Queries(SQL)), is the datatype for "Reg_date" under the
table "Users" VARCHAR() instead of DATE because of the difference in format.
In the excel sheet it is DD/MM/YYYY.
